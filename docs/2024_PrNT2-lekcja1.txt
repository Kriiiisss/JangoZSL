I. Windows
    1. Stwórz odrębne konto Użytkownika Windows
    2. Zainstaluj Git for windows (https://gitforwindows.org/)
    3. Zainstaluj PyCharm Community Edition (https://www.jetbrains.com/edu-products/download/other-PCE.html)
    4. Zainstaluj Notepad++
    5. Zainst Visual Studio Code
    

II. Github
    1. Stwórz konto na GitHubie (o ile nie masz)
    2. Wygeneruj na GitHubie personal access token (classic) i gdzies zapisz
    3. Stwórz nowe repozytorium publiczne

III. Python
    1. Zainstaluj Python 3 (https://www.python.org/downloads/)
